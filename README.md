## 💻 Pré-requisitos

- Expo
- React
- React-Native
- React-Navigation
- @react-navigation/bottom-tabs


## 🚀 Instalando Projeto

Para instalar o projeto, siga estas etapas:

Clone o repositório

```
https://github.com/slashxu/rn-customTabBar.git
```

Baixe as dependências

```
npm install ou yarn install
```

Rodando o app

```
expo start
```
